# -public # 🏥 منظومة نَـبْـضَـة | NABDA CORE SYSTEM
**النبضة المالية الأولى لرواد الأعمال في ليبيا**

## 🚀 نظرة عامة
نظام تقني متكامل لإدارة تسجيل وتمويل المشروعات الناشئة، يعتمد على بروتوكول **"النبضة"** المالي بقيمة **8,000 د.ل** (مقسمة بين رسوم تشغيل وتمويل صافي).

## 🛠️ البنية التقنية (Architecture)
- **المنطقة:** نيويورك (NYC3) - DigitalOcean.
- **المحرك:** OpenAI GPT-4o (عبر GenAI Gateway).
- **قاعدة البيانات:** PostgreSQL (مع نظام Cooldown آلي لمدة 7 أيام).
- **الواجهة:** Glassmorphism UI (Responsive).

## 🔐 صمامات الأمان
1. **بروتوكول الـ 8,000:** توازن حسابي دقيق (800 رسوم + 7,200 تمويل).
2. **الـ Cooldown:** منع تكرار النبضات لنفس الرقم المالي قبل 7 أيام.
3. **التحقق (Validation):** فحص دقيق لأرقام الهواتف والبيانات المدخلة.

## 📁 هيكل المستودع
- `app.js`: المحرك البرمجي وربط الـ API.
- `public/index.html`: واجهة المستخدم.
- `package.json`: المكتبات والمشغلات.
